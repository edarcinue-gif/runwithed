SETUP

Open index.html in a text editor and fill in your Supabase credentials at the top of the <script> tag.

01
Create a free Supabase project
Go to supabase.com → New project → name it "runwithed" → pick any password → create.
02
Run the setup SQL
In your Supabase dashboard → SQL Editor → paste the contents of setup.sql → click Run. This creates all tables and loads all 30 members.
03
Copy your credentials
In Supabase → Settings → API
Copy your Project URL and anon public key.
04
Paste into this file
At the top of the <script> block, replace:
const SUPABASE_URL = 'https://xoxluqaxmhdpicjokici.supabase.co';
const SUPABASE_KEY = 'YOUR_ANON_KEY';
with your real values, then save.
05
Deploy to GitHub Pages
Rename the file to index.html → push to a GitHub repo → enable Pages under Settings → Pages → Branch: main.
